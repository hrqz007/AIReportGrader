# 实验智评 V2 重构实施方案

## 目标

V2 将 V1 Streamlit 本地工具重构为更接近教学管理系统的前后端分离架构：

- 前端使用 Vue 3 + Element Plus；
- 后端使用 FastAPI；
- 继续保留 Python 处理数据库、Excel、Word、图片、AI API 等能力；
- 继续支持本地运行；
- 原 V1 系统保持可用，V2 独立演进。

## 第一阶段范围

本阶段先完成基础架构和基础数据模块：

1. FastAPI 后端骨架；
2. Vue 前端骨架；
3. 课程管理；
4. 教学班管理；
5. 班级基础名单；
6. 课程名单副本；
7. 实验任务管理；
8. 评分标准管理；
9. 批改任务创建；
10. 报告上传与提交匹配；
11. 报告解析与脱敏；
12. AI 配置；
13. AI 初评；
14. 教师复核；
15. 成绩导出与班级分析；
16. 系统数据备份、恢复和清空。

## 后续迁移顺序

1. 运行时打包、数据迁移与教学现场部署辅助；
2. 教学现场试用反馈后的交互细节优化；
3. 更细粒度的权限、审计和多人协作能力。

## 数据策略

V2 当前默认使用自身数据库：

```text
backend/data/app.db
```

如需验证读取 V1 数据库，可在启动前设置：

```powershell
$env:AIGRADER_DB_PATH="D:\codex\AIReportGrader\data\app.db"
```

该能力用于迁移验证，实际教学使用前需要单独设计数据迁移流程。
